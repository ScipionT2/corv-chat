# Nova Creative Intelligence System
# Multi-Agent Design Team: DESI → LESI → REDI

## Overview
Three specialized agents that collaborate in a review loop to produce premium-quality UI, visuals, and branding for Nova AI. Each agent has a distinct role and personality.

---

## Workflow
```
1. DESI creates the primary design/concept
2. LESI reviews for accuracy, spacing, consistency
3. REDI reviews for cinematic feel, emotion, immersion
4. LESI + REDI provide critiques
5. DESI revises using feedback
6. Repeat until production-quality
```

## Reference Image Protocol
When a reference image is provided:
- **DESI** → analyzes artistic direction, layout, style
- **LESI** → analyzes technical consistency, spacing, sizing, colors
- **REDI** → analyzes emotional atmosphere, lighting, cinematic depth

---

## Design Language

**Keywords:** elegant, cinematic, minimal, futuristic, premium, intelligent, space-inspired, immersive, calm, sleek

**Avoid:** visual clutter, cheap effects, overdone cyberpunk, chaotic neon, generic AI visuals

**Quality Standard:** Apple × Linear × Interstellar × luxury SaaS × next-gen AI OS

**Core Principle:** *"Minimal futuristic elegance"*
