❓ Help

Getting Started

Nova is your AI-powered control hub. Use the sidebar to navigate between features.

Chat

Talk to Nova directly. Ask questions, request actions, or have a conversation. Use the action buttons above the input for quick tasks.

Agents

View and manage AI agents. Each agent has a name, model, and system prompt that defines its behavior.

Skills

Skills extend Nova's capabilities. Reload to pick up new skills from the filesystem.

Vision

Upload an image and ask Nova to analyze it using vision-capable models.
